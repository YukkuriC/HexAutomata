      反应核心

      我找到一个方法 - 通过一个固定的村民意识的智能，佐以源自野生生物的"概念" - 创造一类特别的施法物品：反应核心。
      这些法术容器在读写时与普通核心无异，但它们容纳的法术会在特定情形下触发，并从我身上直接抽取媒质以施法。


      每类反应核心在触发时也会发送 iota 至法术的初始栈，其中通常以触发者实体作为栈内首个元素。
      另外，基于我和触发反应核心者之间的量子媒质纠缠效应，内部法术在触发者周围会有一小块额外的施法范围。太便利了 :)

      反应核心内的法术

      利用我和触发者之间的量子媒质纠缠效应，在反应核心触发事件内可以施展一些独特的操作。
      但是，我有一种预感，如果我在核心外施放了这些法术，会发生一些很恐怖的事情。
